# mfanstool Mfans投稿工具前端页面

主要功能：提供第三方视频网站的直链解析，目前已经支持优酷和微博

后端仓库：https://github.com/ChenDoXiu/MfunsBacked

如果你希望添加新的解析服务，欢迎提pr

注意：Youku解析的播放链接是 m3u8 格式，在某些浏览器会触发跨域请求导致视频播放失败（例如 Chrome 浏览器 和 M 站客户端）
